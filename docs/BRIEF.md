Domain selection: Fashion Inspiration

User stories:

1. As a **user**, I want to create an account (name, surname, email) so that I can add likes to garments that I find inspiring.  
2. As a **user**, I want to view a list of garments so that I can browse all available items.  
3. As a user, I want to see which items are trending now.  
4. As a **user**, I want to remove my like (unlike) so that I can delete it from my favorites.

Business rules:  
Each user can like many clothing items.  
Each garment can be liked by many users.  
Each picture is linked to exactly one clothing item.  
Each garment\_like is linked to a specific user and a specific clothing item.  
Each user has a unique email address.  
A user can like a garment only once, they also can remove the like.  
The attributes of garments must be defined.  
If the picture is not present, the name of the garment will be presented instead.  
Each clothing item has exactly one picture linked to it.  
A user may not have any likes.  
A clothing item may not have any likes.
